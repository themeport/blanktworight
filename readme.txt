== Theme Setup ==

= Installation =

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Blank Two Column, Right Sidebar Plugin Requirements ==

Jetpack is an optional plugin to make the most of Blank Two Column, Right Sidebar. You can download Jetpack through your WordPress Dashboard Plugins and search for Jetpack and install. You can also download Jetpack here: https://wordpress.org/plugins/jetpack/

== Setup ==

Blank Two Column, Right Sidebar theme requires only content and your skills to turn it into a better theme. It is meant to help you create your own WordPress theme and or help assist you with other WordPress projects.

== Social Navigation ==

Blank Two Column, Right Sidebar does support social links in the footer. To add them, simply create a new menu in your WordPress dashboard, add custom links to your profile, for example: https://twitter.com/blank_wp. Be sure to choose the 'social' display option and save any changes. Our themes support most popular social networks.
